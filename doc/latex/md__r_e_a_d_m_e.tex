Este proyecto de Programación I permite jugar dos juegos clásicos en consola\+: {\bfseries{Batalla Naval}} y {\bfseries{\doxylink{class_ahorcado}{Ahorcado}}}, desarrollados en Java. Ambos juegos están diseñados para dos jugadores, alternando turnos, validación de entradas y condiciones de victoria.\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Ejecucion del programa}{Ejecucion del programa}}\label{md__r_e_a_d_m_e_autotoc_md1}

\begin{DoxyEnumerate}
\item Abrí el proyecto en tu IDE (Blue J, Visual Studio, etc).
\item Ejecutá la clase \doxylink{_main_8java}{Main.\+java}.
\item Aparecerá un menú con tres opciones\+:
\begin{DoxyItemize}
\item 1. \doxylink{class_ahorcado}{Ahorcado}
\item 2. Batalla Naval
\item 3. Salir
\end{DoxyItemize}
\item Ingresá el número del juego que querés jugar y seguí las instrucciones en consola.
\end{DoxyEnumerate}

\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysection{\texorpdfstring{Juegos incluidos}{Juegos incluidos}}\label{md__r_e_a_d_m_e_autotoc_md3}
\hypertarget{md__r_e_a_d_m_e_autotoc_md4}{}\doxysubsection{\texorpdfstring{Ahorcado}{Ahorcado}}\label{md__r_e_a_d_m_e_autotoc_md4}

\begin{DoxyItemize}
\item Jugador 1 escribe una palabra secreta.
\item Jugador 2 intenta adivinarla, letra por letra.
\item Tiene 6 oportunidades de error antes de perder.
\item Luego se invierten los roles.
\item Se pueden jugar múltiples rondas y se muestra un resumen de fallos y quién ganó.
\item Al finalizar las rondas se puede reiniciar el juego con nuevos jugadores.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md5}{}\doxysubsection{\texorpdfstring{Batalla Naval}{Batalla Naval}}\label{md__r_e_a_d_m_e_autotoc_md5}

\begin{DoxyItemize}
\item Cada jugador coloca 4 barcos de diferentes tamaños.
\item Los barcos se colocan en un tablero de 5x5, eligiendo dirección (horizontal o vertical).
\item Durante el juego\+:
\begin{DoxyItemize}
\item Si acertás el disparo (Tocado!), jugás de nuevo.
\item Si fallás (Agua!), el turno pasa al otro jugador.
\end{DoxyItemize}
\item El juego termina cuando un jugador hunde todos los barcos del otro.
\item Podés jugar varias partidas seguidas si así lo deseás.
\end{DoxyItemize}
